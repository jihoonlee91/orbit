# Phase 6-9: Promotional Thumbnail

## Goal

Make shared Orbit links recognizable as an action arcade game before the viewer
opens the site.

## Composition

- Use a 1200×630 landscape canvas for common social preview cards.
- Center the player, vertical harpoon, splitting orb, and `ORBIT` title so the
  core action remains readable in small or cropped previews.
- Frame the action with world-tour scenery and the hidden finale's eclipse,
  using neon cyan, magenta, orange, and gold against deep navy.
- Do not reproduce menus, buttons, HUD elements, or small instructional text.

## Delivery

- Store the optimized final artwork at `public/og-image.jpg`.
- Keep Open Graph and Twitter metadata pointed at the same public asset.
- Verify the generated production build includes the image at 1200×630.
